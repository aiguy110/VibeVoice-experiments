VibeVoice Experiments

This page showcases audio samples generated using Microsoft's VibeVoice text-to-speech model. The experiments include:

  • Official 1.5B model with float32 parameters
  • Unofficial half-precision (f16) versions of both 1.5B and 7B models with weights converted directly from the official f32 weights

Each voice has reference samples and synthesized outputs from different model configurations.
